# power-bi
This project focuses on the design and development of an interactive dashboard using Power BI to analyze and visualize a given dataset. The objective was to extract meaningful insights and support data-driven decision-making through effective visualization techniques.

The dataset was first explored and preprocessed by identifying key dimensions and measures, handling missing values, and removing irrelevant or redundant columns to improve data quality and performance. Important variables such as year, work status, income, region, and prestige were retained for analysis.

The dashboard includes multiple visualizations such as a time-series line chart to analyze trends over time, bar charts for category-wise and region-wise comparisons, a histogram to understand the distribution of prestige scores, and a scatter plot to study the relationship between income and prestige. Interactive features like slicers, drill-down functionality, and tooltips were implemented to enhance user experience and allow dynamic data exploration.

Key insights derived from the dashboard highlight patterns in prestige levels across different categories and regions, as well as a positive relationship between income and prestige. Performance optimization techniques such as data reduction and efficient aggregation were applied to ensure smooth functioning of the dashboard.

This project demonstrates the effective use of Power BI for data visualization, analysis, and interactive reporting.
